# CHS (Community Hub System) 문서

> 사람을 모으는 커뮤니티 허브 시스템 — SSOO 모노레포 3번째 앱

## 개요

CHS는 **일(PMS) + 문서(DMS) + 사람(CHS)** 삼각 완성형 업무 허브의 마지막 축입니다.

| 기능 | 설명 |
|------|------|
| **SNS 피드** | LinkedIn 스타일 타임라인, 게시물, 반응, 멘션 |
| **게시판** | 공지사항, Q&A, 자유게시판, 인력 모집 |
| **인력풀** | 스킬맵, 프로젝트 이력, 전문가 검색, 추천 |
| **소셜** | 팔로우, 북마크, 알림 |

## 기술 스택

- **프론트엔드**: Next.js 15 + React 19 + TypeScript 5 + Tailwind CSS 3
- **UI**: shadcn/ui (Radix) — PMS와 동일 컴포넌트, 틸 색상 테마
- **상태**: Zustand 5 + TanStack Query 5
- **백엔드**: NestJS (모듈러 모놀리스, `modules/chs/`)
- **DB**: PostgreSQL `chs` 스키마 (`ch_` 접두사)
- **포트**: 3002

## 3앱 색상 패밀리

| 앱 | 색상 | Hue | 의미 |
|---|---|---|---|
| PMS | 네이비 `#0A1E5A` | 213° | 업무·신뢰 |

| **CHS** | **틸 `#0A3D3D`** | **180°** | **연결·성장** |
