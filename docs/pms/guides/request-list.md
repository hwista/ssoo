# 요청 목록 가이드

> 최종 업데이트: 2026-02-10

요청 목록 화면의 조회/검색/상세 패널 사용 방법을 설명합니다.

---

## 목적

- 프로젝트 요청 상태의 목록을 조회한다.
- 검색 조건으로 요청을 필터링한다.
- 행 선택 시 세컨 그리드로 상세를 확인한다.

---

## 화면 구성

- 메인 그리드: 프로젝트 공통 테이블 기준 목록
- 세컨 그리드: 행 선택 시 하단 플로팅 상세 패널

---

## 조회 조건

- 상태: `request` (요청)
- 단계: `waiting`, `in_progress`, `done`
- 고객사 ID
- 프로젝트명(검색)

---

## 세컨 그리드

- 행 클릭 시 플로팅 패널이 열린다.
- 요청 상세 테이블 컬럼에 맞춘 데이터가 표시된다.
- 패널 상단 중앙의 쉐브론 버튼으로 접기/펼치기 가능
- 데이터가 없을 때도 그리드 영역 높이를 유지한다.
- 패널 너비는 메인 그리드와 동일하며 높이는 메인 그리드의 2/3 정도다.
- 쉐브론 버튼은 DMS 사이드카와 동일한 스타일로 표시된다.
- 패널은 아래로 접히고 위로 펼쳐지는 슬라이드 애니메이션이 적용된다.
- 패널 데이터 영역 배경은 사이드바 색상과 동일하다.
- 패널 배경과 컬럼 헤더 배경 색상은 서로 스왑된다.
- 행 배경은 줄무늬로 구분되며 선택 행은 사이드바 활성 색상으로 강조된다.
- 메인/서브 그리드 헤더 배경 색상은 동일하게 맞춘다.
